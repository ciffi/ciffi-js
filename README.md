# CiffiDesign Frontend Generator #

#### installare globalmente tramite npm
```
npm install -g ciffi
```
#### setup new project
```
ciffi setup projectname
```
#### local dev
```
ciffi dev
```
#### generate build
```
ciffi build
```
#### new page html and js
```
ciffi newpage pagename
```
#### new js module
```
ciffi newmodule modulename
```
