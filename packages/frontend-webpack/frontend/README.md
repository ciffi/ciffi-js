# Ciffi Frontend Base Project #

#### local dev
```
npm start
```
or
```
npm run dev
```
#### generate build
```
npm run build [env]
```
#### test code
```
npm test
```
#### regenerate config.js
```
npm run config [env]
```
#### copy assets only
```
npm run assets
```

## WORKFLOW

1. run dev task and start to write code
2. run build task and you are ready to deploy
