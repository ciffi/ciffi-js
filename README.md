# CiffiDesign Frontend Generator #

##installare globalmente tramite npm##

```

npm install -g ciffi

```

##setup e configurazione##

```

ciffi setup nomeprogetto

```

##sviluppo locale##

```

ciffi dev

```

##build##

```

ciffi build

```

##nuova pagina##

```

ciffi newpage nomepagina

```

##nuovo modulo##

```

ciffi newmodule nomemodulo

```
