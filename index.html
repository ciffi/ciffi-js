CiffiDesign Frontend Generator

install ciffi

npm install -g ciffi

setup new project

ciffi setup projectname

local dev server

ciffi serve

local dev build

ciffi dev

generate build

ciffi build

new page html and js

ciffi newpage pagename

new js module

ciffi newmodule modulename
