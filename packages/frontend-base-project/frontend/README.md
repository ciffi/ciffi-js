# Ciffi Frontend Base Project #

#### local dev
```
npm run dev
```
or
```
ciffi dev
```
#### generate build
```
npm run build [env]
```
or
```
ciffi build[:env]
```

## WORKFLOW

1. run dev task and start to write code
2. run build task and you are ready to deploy
